# UIRouter for KCM

## Repository Setup Instructions

### 1. Create GitHub Repository
- Repository name: `uirouter-for-kcm`
- Description: "Type-safe routing library for Kotlin Compose Multiplatform"
- Public repository
- Add topics: `kotlin`, `compose-multiplatform`, `android`, `ios`, `routing`, `navigation`

### 2. Update Build Configuration
Before publishing, update the following in `build.gradle.kts`:
```kotlin
group = "com.github.YOUR_GITHUB_USERNAME"
```

And in `library/build.gradle.kts`, update the developer info:
```kotlin
developer {
    id.set("YOUR_GITHUB_USERNAME")
    name.set("Your Name")
    email.set("your.email@example.com")
}
```

### 3. Initialize Git and Push
```bash
git init
git add .
git commit -m "Initial commit: UIRouter for KCM v1.0.0"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/uirouter-for-kcm.git
git push -u origin main
```

### 4. Create Release Tag
```bash
git tag -a v1.0.0 -m "Release v1.0.0"
git push origin v1.0.0
```

### 5. Publish to JitPack
1. Visit https://jitpack.io
2. Enter: `https://github.com/YOUR_USERNAME/uirouter-for-kcm`
3. Click "Look up"
4. Click "Get it" for version v1.0.0
5. Wait for build to complete

### 6. Update README.md
Replace all instances of `yourusername` with your actual GitHub username in:
- README.md
- CHANGELOG.md
- All documentation links

### 7. Test Installation
Create a test project and add:
```kotlin
repositories {
    maven { url = uri("https://jitpack.io") }
}

dependencies {
    implementation("com.github.YOUR_USERNAME:uirouter-for-kcm:1.0.0")
}
```
